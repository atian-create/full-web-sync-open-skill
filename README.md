# Full Web Sync Open Skill

Package a finished Skill, local project, or open-source repo into a public launch bundle.

This Skill helps an agent turn a local project into:

- repo-facing GitHub docs
- release notes
- public repo verification
- X platform launch text
- Knowledge Planet community copy
- optional internal archive notes

It is not a reposting bot, traffic-growth tool, or viral-copy machine.

## Install

Copy this folder into your Codex skills directory:

```text
~/.codex/skills/full-web-sync-open-skill/
```

Then ask Codex:

```text
Use $full-web-sync-open-skill to package this Skill for GitHub, X, and Knowledge Planet.
```

## Typical Use

Use it after a project is already mostly finished:

- a new Skill
- a local tool
- a public GitHub repo
- a template project
- a Build in Public release
- a creator workflow package

The Skill will inspect the project, keep facts accurate, avoid private details, update or create repo-facing materials, and write platform-ready launch copy.

## Default Deliverables

- GitHub repo or release status
- README / Skill doc update guidance
- X text that can be copied directly
- Knowledge Planet post that can be copied directly
- verification notes

It does not write Xiaohongshu or WeChat public account copy by default.
